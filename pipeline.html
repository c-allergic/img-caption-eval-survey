CapsBench Evaluation Pipeline

200 images · 2,471 yes-no questions · 17 evaluation categories

Core Dataset

  • • 200 hand-curated images
  • • 2,471 yes-no question pairs
  • • ~12 questions per image avg.
  • • 17 fine-grained categories
  • • 9 visual domains covered

Relation to DPG-Bench

  • • DPG-Bench: text prompt → image → VQA
  • • CapsBench (reversed): image → caption → QA
  • • DPG-Bench Hard: GPT-4o caption → QA
  • • Both: "parallels how we constructed CapsBench"

Key Findings

  • • PG Captioner scores highest (72.19%)
  • • All models miss entity shape, size, artifacts
  • • Caption length ≠ quality
  • • QA ambiguity needs ablation study
